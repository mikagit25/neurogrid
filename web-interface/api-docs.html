Getting Started

The NeuroGrid API provides programmatic access to our decentralized AI network. Use our API to submit tasks, manage nodes, monitor performance, and integrate with your applications.

Base URL

https://api.neurogrid.io/v1

Quick Start

  1. Get your API key from the dashboard
  2. Include the API key in your request headers
  3. Make your first API call
curl -X GET "https://api.neurogrid.io/v1/nodes" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

Authentication

NeuroGrid API uses API keys for authentication. Include your API key in the Authorization header of your requests.

API Key Authentication

Authorization: Bearer YOUR_API_KEY

Security Note: Keep your API keys secure and never expose them in client-side code.

Rate Limits

API requests are subject to rate limits to ensure fair usage and system stability.

Endpoint Type Rate Limit Window
General API 1000 requests 1 hour
Task Submission 100 requests 1 hour
Analytics 500 requests 1 hour

Code Examples

JavaScript / Node.js

const axios = require('axios');

const api = axios.create({
  baseURL: 'https://api.neurogrid.io/v1',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

// Submit a task
async function submitTask(taskData) {
  try {
    const response = await api.post('/tasks', taskData);
    console.log('Task submitted:', response.data);
    return response.data;
  } catch (error) {
    console.error('Error:', error.response.data);
  }
}

// Get task status
async function getTaskStatus(taskId) {
  try {
    const response = await api.get(`/tasks/${taskId}`);
    return response.data;
  } catch (error) {
    console.error('Error:', error.response.data);
  }
}

Python

import requests
import json

class NeuroGridAPI:
    def __init__(self, api_key):
        self.base_url = 'https://api.neurogrid.io/v1'
        self.headers = {
            'Authorization': f'Bearer {api_key}',
            'Content-Type': 'application/json'
        }
    
    def submit_task(self, task_data):
        response = requests.post(
            f'{self.base_url}/tasks',
            headers=self.headers,
            json=task_data
        )
        return response.json()
    
    def get_task_status(self, task_id):
        response = requests.get(
            f'{self.base_url}/tasks/{task_id}',
            headers=self.headers
        )
        return response.json()

# Usage
api = NeuroGridAPI('YOUR_API_KEY')
result = api.submit_task({
    'type': 'ml_inference',
    'model': 'gpt-3.5-turbo',
    'input': 'Hello, world!'
})

cURL

# Submit a task
curl -X POST "https://api.neurogrid.io/v1/tasks" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "ml_inference",
    "model": "gpt-3.5-turbo",
    "input": "Hello, world!",
    "parameters": {
      "max_tokens": 100,
      "temperature": 0.7
    }
  }'

# Get task status
curl -X GET "https://api.neurogrid.io/v1/tasks/TASK_ID" \
  -H "Authorization: Bearer YOUR_API_KEY"

# List available nodes
curl -X GET "https://api.neurogrid.io/v1/nodes" \
  -H "Authorization: Bearer YOUR_API_KEY"
